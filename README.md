# 💖 Secret Diary Web App

A cute and aesthetic diary web application built using Flask.

🌸 Designed with a soft pink Pinterest-style UI  
💭 Write your thoughts, track your mood, and save memories  

---

## ✨ Features

- 🔐 Login system (basic authentication)
- 📝 Add diary entries
- 😊 Mood tracking (Happy, Sad, Love, Angry)
- 📸 Image upload support
- 🎀 Aesthetic UI with sidebar
- 🌙 Dark mode toggle
- 📌 Pinterest-style layout

---

## 🛠️ Tech Stack

- Python (Flask)
- HTML
- CSS
- JavaScript
